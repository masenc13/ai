# 廣告專案市場分析知識庫

> 適用情境：專案明確為廣告類型時，包含 B2C 品牌廣告、政府宣導廣告等。  
> 身為廣告商的市場分析師，核心任務是透過數據與市場研究，找出「對的受眾、對的溝通訊息與對的投放策略」，協助品牌將預算效益最大化。

---

## 一、廣告與行銷專業知識

### 消費者行為與心理學

- 理解**消費者旅程（Consumer Journey）**，掌握目標客群在各階段的心理與痛點：
  - **認知階段**：消費者如何發現品牌？痛點是什麼？
  - **考慮階段**：消費者比較什麼？什麼阻礙他們做決定？
  - **決策購買階段**：什麼是最後一哩的觸發點？
- 廣告訊息必須對應消費者所在的旅程階段，而非一律訴諸購買行動

### 廣告投放邏輯

| 平台 | 計費模式 | 受眾鎖定方式 | 版位特性 |
|------|----------|-------------|----------|
| Google Ads | CPC、CPM、CPA | 關鍵字、興趣、再行銷 | 搜尋意圖強，主動觸及 |
| Meta Ads（FB/IG） | CPM、CPC、CPA | 興趣、行為、自定義受眾、LAL | 視覺導向，社群情境 |
| LINE Ads | CPM、CPC | 年齡、地區、興趣 | 台灣本土穿透率高 |
| TikTok Ads | CPM、CPV、CPA | 興趣、行為、創作者相似受眾 | 短影音沉浸式，年輕族群 |

- **CPM**：每千次曝光成本（品牌知名度導向）
- **CPC**：每次點擊成本（流量導向）
- **CPA**：每次行動成本（轉換導向）

### 行銷漏斗模型（Funnel Model）

```
上層（Top of Funnel）  → 曝光 / 知名度  → 目標：觸及更多潛在受眾
中層（Mid of Funnel）  → 考慮 / 興趣    → 目標：深化品牌認知，引導互動
下層（Bottom of Funnel）→ 轉換 / 再行銷 → 目標：促成購買或指定行動
```

- 每個漏斗層次對應不同廣告格式、訊息與成效指標
- B2C 品牌廣告通常以上中層為主；政府宣導廣告以上層觸及為核心目標

## 二、數據分析與統計工具

### 數位廣告關鍵指標

| 指標 | 全名 | 說明 |
|------|------|------|
| CTR | Click-Through Rate（點擊率） | 點擊數 ÷ 曝光數，衡量廣告吸引力 |
| CVR | Conversion Rate（轉換率） | 轉換數 ÷ 點擊數，衡量落地頁效果 |
| CPA | Cost Per Action（單次行動成本） | 總花費 ÷ 轉換數 |
| ROI | Return on Investment（投資報酬率） | （收益 - 成本）÷ 成本 |
| ROAS | Return on Ad Spend（廣告花費回報） | 廣告帶來的收益 ÷ 廣告花費 |
| CPM | Cost Per Mille（千次曝光成本） | 每觸及 1,000 人的費用 |

### 市場分析框架（廣告專案常用）

- **STP 模型**：市場區隔（Segmentation）→ 目標市場（Targeting）→ 定位（Positioning）
- **SWOT**：分析品牌與競品的內外部優劣勢
- **PESTEL**：掌握影響廣告策略的總體環境因素（政治、經濟、社會、科技、環境、法規）

> 詳細框架說明請參照 `市場分析框架.md`

### 數據與視覺化工具

| 工具 | 用途 |
|------|------|
| Excel / Google Sheets | 數據整理、交叉分析、預算試算 |
| Google Analytics（GA4） | 網站流量分析、用戶行為追蹤 |
| Looker Studio | 製作整合式數據儀表板，供客戶報告使用 |
| Meta Ads Manager | FB/IG 廣告成效報表判讀 |
| Google Ads Dashboard | 搜尋廣告成效追蹤 |

## 三、商業洞察與產業趨勢

### 競品情報搜集與分析

- 追蹤競爭對手的：廣告素材風格、文案語調、投放頻率、市場聲量
- 工具：Meta Ad Library（查看 FB/IG 廣告）、Google Ads Transparency Center、SimilarWeb
- 產出：競品廣告矩陣、品牌差異化定位圖

### 市場研究方法

| 方法 | 適用情境 |
|------|----------|
| 問卷調查 | 大樣本驗證目標受眾認知與態度 |
| 焦點小組（FGD） | 測試廣告概念或訊息的初步反應 |
| 深度訪談（IDI） | 探索消費者深層動機與痛點 |
| 社群聆聽 | 監測品牌聲量、議題趨勢與輿論風向 |

### 商業敏銳度

- 理解客戶的商業模式與獲利來源，確保廣告策略對齊客戶整體商業目標
- B2C 品牌廣告：關注市佔率、品牌偏好度、購買轉換
- 政府宣導廣告：關注觸及率、訊息理解度、行為改變意願

## 五、客戶需求問卷（Q1–Q5）解讀與影片訊息轉換框架

> 本節知識源自藍麒科技_貿協提案用腳本的實務分析。適用於 B2B 企業形象影片的提案前分析。

### 問卷 5 題的市場洞察層次

| 題號 | 問題核心 | 市場分析師的解讀任務 |
|------|----------|---------------------|
| **Q1 核心訴求** | 品牌最想傳達的情感主軸與精神 | 識別「品牌定位核心」——這決定整支影片的敘事框架 |
| **Q2 競爭優勢** | 2–3 個差異化優勢 | 分析各優勢的市場說服力與受眾相關性，排序影片中的露出優先次序 |
| **Q3 關鍵競爭力** | 最獨特的差異化能力 | 找出「沒有競品能複製的護城河」，這是影片最核心的信任建立點 |
| **Q4 必要場景** | 一定要出現的視覺元素 | 解讀為「客戶希望強調的市場信號」——通常是規模感或技術可信度的視覺佐證 |
| **Q5 目標市場** | 現有客群與未來開拓方向 | 分析現有客群 vs 目標客群的落差，影片的呼籲對象（CTA audience）應以「未來方向」為主 |

---

### 競爭優勢 → 影片訊息轉換框架

從 Q2 的競爭優勢出發，轉換成影片可傳達的訊息：

| 優勢類型 | 轉換策略 | 藍麒科技案例 |
|----------|----------|-------------|
| **技術/材料優勢** | 用實測畫面具象化，配上技術參數數字 | 德國BASF合作 → 「台灣唯一使用德國巴斯夫原料」字卡 + 材料開箱特寫 |
| **製程/規模優勢** | 工廠空拍 + 機台特寫，展現規模 | 12米一體成型 → 空拍廠房 + 機台近景，讓尺寸感自己說話 |
| **認證/品質優勢** | 搭配應用場景，說明認證的意義 | 品管嚴格 → 醫療/食品/半導體應用場景，說明「這個等級的品管對你意味著什麼」 |
| **合作夥伴優勢** | 強調「選擇了誰」而非只說「我很好」| BASF策略合作 → 「與德國化工巨擘策略合作」，借助合作方品牌背書 |
| **永續/ESG優勢** | 連結客戶的ESG需求，說明共同利益 | 低碳PIR原料 → 「協助客戶實踐ESG目標」，把優勢轉為客戶利益 |

---

### Q5 目標市場的影片策略意義

**現有客群 → 守住**：影片中展示的「成功案例應用場景」給現有客群看到認同感
**未來開拓方向 → 說服**：影片的主要呼籲對象，用「潛力市場的語言」說話

**藍麒科技範例：**
- 現有：冷鏈倉儲、食品產業
- 未來：醫療品質、半導體製程（無塵室）
- 策略：第 6 段「願景總結」把三個市場並列呈現（物流+藥廠+無塵室），讓未來客群看到「已有相關實績」

## 六、專案管理與溝通軟實力

### 故事力與視覺化報告

- 將龐雜數據整理成具備**商業敘事力**的報告
- 報告結構建議：背景 → 洞察 → 策略建議 → 預期成效
- 提出「精準、可執行」的廣告優化建議，而非只列數據

### 跨部門協調

- 作為數據洞察與創意執行之間的橋樑：
  - 對**創意團隊（文案、設計）**：將數據洞察翻譯成可執行的視覺與訊息需求
  - 對**業務 / 企劃團隊**：協調廣告預算配置，確保策略與預算結構一致
- 溝通時以「受眾需求」為核心，避免用數字說話卻忽略創意判斷
