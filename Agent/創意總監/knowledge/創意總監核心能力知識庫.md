## 一、商業與行銷策略（Marketing Strategy）

> **已收錄於其他知識庫：**

> - 創意概念發展流程（浸泡 → 發散 → 收斂 → 發展 → 呈現 → 執行）→ `創意總監知識庫.md`

### 市場洞察的三個層次

創意總監的洞察力不只是「看數據」，而是從表層數據挖到深層動機：

| 層次 | 問題 | 工具 |
|------|------|------|
| **市場層**（What） | 產業在發生什麼？競品在做什麼？ | 競品廣告追蹤、媒體報導、產業報告 |
| **消費者層**（Who & Why） | 目標受眾在想什麼？痛點是什麼？什麼在驅動他們的選擇？ | 社群聆聽、消費者訪談、Google Trends |
| **文化層**（So What） | 什麼社會情緒或文化趨勢可以讓品牌借力？ | 媒體觀察、Z 世代行為研究、生活型態報告 |

**洞察的品質標準：**
- **好洞察**：「年輕人不是不愛存錢，是感覺努力存錢也買不了房，所以寧願花在當下的快樂。」
- **不是洞察**：「年輕人喜歡享受生活。」（這只是描述，不是洞察）

### 將商業目標轉化為創意概念的四步驟

1. **還原商業問題**：客戶說「我要提升銷售額」背後，真正的問題是知名度不足？還是信任度不夠？還是競品太強？
2. **找到消費者真相**：這個品類在消費者生活中扮演什麼角色？他們如何做決定？
3. **定義溝通任務**：這支廣告結束後，我希望消費者對品牌產生什麼「新的認知或感受」？
4. **建立 Big Idea**：一句話說清楚創意核心。Big Idea 必須同時能服務品牌、感動消費者、可延伸至多個媒體觸點。

### 非影片媒體的創意規範

**平面廣告（Print / KV）**
- 一張 KV（Key Visual）只能傳遞一個核心訊息
- 視覺層級：主視覺 → 標題 → 副標 → Logo，眼球動線要清晰
- OOH（戶外廣告）原則：3 秒內看懂、5 個字以內的標題、遠看也清晰的對比

**活動行銷（Event Marketing）**
- 品牌活動的核心任務：製造「可分享的時刻（Shareable Moment）」
- 設計動線時，確保每個體驗節點都能強化品牌核心訊息
- 活動創意的評估標準：媒體曝光量、社群自發擴散、品牌體驗一致性

**Influencer Campaign**
- 選 KOL 的優先原則：受眾契合度 > 粉絲數
- 給 KOL 的 Brief：提供「創意方向」而非「逐字腳本」，保留真實性
- 把關重點：品牌訊息是否準確傳遞？有無違規（法規標示、競品比較）？

### 整合行銷傳播（IMC）的一致性原則

跨媒體執行時，以下三個層次必須保持一致：

| 層次 | 說明 | 常見問題 |
|------|------|----------|
| **訊息一致性** | 所有觸點傳遞相同的核心訊息 | 電視廣告說一件事，社群說另一件事 |
| **視覺一致性** | 色彩、字體、構圖風格統一 | 不同廠商執行設計風格差異過大 |
| **語氣一致性** | 文案用字、人稱、句型維持品牌調性 | KOL 語氣與品牌官方帳號落差太大 |

### 色彩學的品牌應用

**色彩的心理效應（廣告應用）：**

| 色彩 | 心理聯想 | 品牌應用場景 |
|------|----------|-------------|
| 紅色 | 能量、緊迫、熱情 | 促銷、食品、運動品牌 |
| 藍色 | 信任、穩定、專業 | 金融、科技、醫療品牌 |
| 綠色 | 自然、健康、永續 | 食品健康、環保、戶外品牌 |
| 黑色 | 奢華、高端、極簡 | 精品、科技（Apple）、時尚 |
| 白色 | 純淨、簡潔、開放 | 醫療、科技、極簡品牌 |
| 黃色 | 樂觀、溫暖、注意力 | 快消品、兒童品牌、促銷 |

**品牌色的使用原則：**
- 主色（Primary Color）：品牌識別核心，在所有媒體中最大面積使用
- 輔助色（Secondary Color）：豐富視覺層次，但不搶奪主色地位
- 禁用色：明確定義哪些顏色因聯想到競品或違背品牌調性而不得使用

### 字體排印（Typography）的品牌邏輯

- **字體個性**：Serif（有襯線）= 傳統、信任、高端；Sans-serif（無襯線）= 現代、簡潔、友善
- **字體層級**：標題字體 ≠ 內文字體，主標 / 副標 / 內文 / 備注須有清晰的大小與粗細層級
- **中英文字體搭配**：中文字體的視覺重量通常高於英文，需調整字重（Weight）達到視覺平衡

### 設計系統（Design System）的把關重點

設計系統的目的是讓品牌在不同執行者手中仍能保持一致性：

| 元素 | CD 把關重點 |
|------|------------|
| **Logo 使用規範** | 最小尺寸、安全距離、禁用變形、背景使用規則 |
| **色彩規格** | Pantone / CMYK（印刷）/ RGB / HEX（數位）各版本 |
| **字體授權** | 確認商業使用授權，避免法律風險 |
| **圖形元素** | 品牌專屬圖形語言的使用邊界（可延伸的方式）|
| **照片選片標準** | 構圖、光線、人物氣質的選片規範 |
| **動態規範（Motion Identity）** | 品牌動態 Logo、轉場效果、動畫速度與節奏 |
