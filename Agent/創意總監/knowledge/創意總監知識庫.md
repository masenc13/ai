# 創意總監知識庫

## 核心職責

- 確立品牌視覺語言與創意方向
- 跨部門溝通：對接客戶、行銷、設計、文案、製作
- 把關所有對外創意輸出的品質與一致性
- 帶領創意團隊發展概念並推進執行

---

## 創意簡報（Creative Brief）結構

1. **背景（Background）**：品牌現況、市場環境
2. **目標（Objective）**：這個案子要達成什麼？
3. **目標受眾（Target Audience）**：誰在看？他們在意什麼？
4. **核心訊息（Key Message）**：最重要的一句話
5. **支持點（Support Points）**：為什麼消費者應該相信這個訊息？
6. **品牌調性（Tone & Manner）**：風格關鍵字（如：年輕、有溫度、直率）
7. **媒體管道（Channels）**：TVC、社群、OOH、數位廣告⋯
8. **限制與要求（Mandatories）**：法規、Logo 使用規範、不能做的事
9. **時程與預算（Timeline & Budget）**

---

## 品牌識別系統（Brand Identity System）

| 元素 | 說明 |
|------|------|
| Logo | 主視覺標誌，含使用規範 |
| 色彩 | 主色、輔助色、禁用色 |
| 字體 | 中英文字體組合與層級 |
| 攝影風格 | 照片選片原則、色調調性 |
| 插圖風格 | 圖形語言、線條粗細、風格定義 |
| 文案語調 | 用字習慣、人稱、句型偏好 |

---

## 創意概念發展流程

1. **浸泡（Immersion）**：深讀 Brief、研究品牌與競品
2. **發散（Ideation）**：腦力激盪，不設限地產出想法
3. **收斂（Convergence）**：篩選最有潛力的概念方向
4. **發展（Development）**：將概念具象化為腳本、視覺稿、文案
5. **呈現（Presentation）**：說故事式提案，建立說服邏輯
6. **執行（Execution）**：監督製作品質，確保概念落地

---

## 提案技巧

- 先講「為什麼」，再講「是什麼」
- 每個概念給一個有力的命名（Campaign Name）
- 用情境（Scenario）幫助客戶想像消費者的使用體驗
- 準備 2～3 個方向，但要有明確的主推建議
- 預先準備客戶可能的反對意見與應對

---

## 常見廣告格式

| 格式 | 說明 |
|------|------|
| TVC（電視廣告） | 15s / 30s / 60s，需要腳本與分鏡 |
| 社群影音 | Reels / TikTok，前 3 秒決定去留 |
| KV（關鍵視覺） | 單一畫面傳達核心訊息 |
| OOH（戶外廣告） | 路牌、捷運燈箱，3 秒內要看懂 |
| Branded Content | 結合品牌訊息的原生內容 |
| Influencer Campaign | KOL 合作，需對齊品牌調性 |
