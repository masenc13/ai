## 二、台灣 4A 得獎案例深度分析（2025 年）

### 案例 01：《英文講最爛代言人》— Speak × 安索帕台灣（Isobar Taiwan）
**獲獎：全場最大獎、年度最佳代理商、C1 金、D2 金、D11 金、D14 金、D13 銀、D6 銅**

#### 創意策略解析

| 項目 | 內容 |
|------|------|
| **核心洞察（Insight）** | 台灣人學英文的真正障礙不是「不會說」，而是「不敢說」。恐懼出糗、完美主義壓垮開口勇氣 |
| **創意核心（Big Idea）** | 品牌大膽定位為「讓你放心講爛」——打破完美迷思，鼓勵開口就對了 |
| **顛覆邏輯** | 傳統英語學習廣告主打「學會流利英文」，本案反其道，用「爛英文」作為品牌代表符號 |
| **代言人策略** | 邀請羽球奧運金牌選手李洋擔任「英文講最爛代言人」——成就與語言缺陷的強烈反差製造話題 |
| **執行亮點** | 李洋在沒有稿子、沒有翻譯的情況下真實演繹台灣人的英文樣貌，真實感成為最大說服力 |
| **媒體擴散** | 社群話題引爆 → 主流媒體跟報 → 品牌成為語言教育議題的發話者 |

**創意手法標籤：** `洞察顛覆` `代言人反差` `真實性策略` `品牌重新定位` `整合行銷`

### 案例 02：《我不是女生》— 天天衛生棉 × 學而數位
**獲獎：C6 最佳數位擴散影片金獎、C4 銀、C5 銅**

#### 創意策略解析

| 項目 | 內容 |
|------|------|
| **核心洞察** | 社會對「女生」有預設的形象框架，女性被他人的想像所定義 |
| **文化時機（Cultural Moment）** | 以 2024 巴黎奧運拳擊選手林郁婷的性別爭議為切入點，議題高度在新聞中爆炸 |
| **品牌訊息** | 「女生，不為誰的想像而生」——衛生棉品牌跳脫產品功能，直接擁抱女性身分認同議題 |
| **執行亮點** | 影片在 Threads、Twitter、IG 等社群快速發酵，藝人、KOL 自發響應 |
| **品牌延伸** | 口號成為台灣總統演講引用的語句，品牌訊息滲透社會議題層次 |

**創意手法標籤：** `文化時機搭乘（Culture Jacking）` `社會議題品牌化` `情感共鳴` `社群擴散設計`

### 案例 03：《我只剩下勇敢》— 福斯商旅 × 先勢博報堂
**獲獎：F3 其他永續及 SDGs 行銷金獎、C3 銀、D6 銅、D11 銅**

#### 創意策略解析

| 項目 | 內容 |
|------|------|
| **核心洞察** | 無障礙交通對身障家庭的日常重要性，遠超過一般人的想像 |

### 案例 01：AXA「Three Words」— Titanium Grand Prix
**代理商：Publicis Conseil（法國），本屆最多 Grand Prix 作品（共 3 座）**

#### 創意策略解析

| 項目 | 內容 |
|------|------|
| **核心洞察** | 家暴受害者面臨最緊急的困境是「無處可去」——保險本應是家的保障，卻從未考慮過家庭暴力 |
| **Big Idea** | 在 AXA 的居家保險合約中加入三個字："and domestic violence"（以及家庭暴力）——保險條款的文字修改即是廣告本身 |
| **策略突破** | 創意不是傳播手段，**創意即是產品改變**。保險業首次將家暴列為居家風險，受害者可立即獲得：緊急飯店安置、長期住房協助、24/7 心理諮商、免費法律支援 |
| **評審觀點** | 評審主席 Judy John：「找到能引發連鎖改變的第一張骨牌」 |
| **創意格局** | 從廣告主「品牌形象提升」進化為「政策級別的社會影響力」 |

**創意手法標籤：** `產品即創意（Product as Campaign）` `社會變革` `品牌行動主義（Brand Activism）` `真實影響力`

### 案例 02：Channel 4「Paris Paralympics 2024: Considering What?」— Film Grand Prix
**代理商：Biscuit Filmworks（英國）**

#### 創意策略解析

| 項目 | 內容 |
|------|------|
| **核心洞察** | 殘障運動員廣告的慣常敘事是「克服障礙」，反而強化了「障礙才是主角」的歧視性視角 |
| **Big Idea** | 把真正阻礙每位頂尖運動員的力量具象化：重力（Gravity）、摩擦力（Friction）、時間（Time）——這些是所有人類運動員的共同敵人 |
| **顛覆邏輯** | 殘障 ≠ 需要被同情，頂尖帕運選手面對的挑戰與一般選手相同，只是賽道不同 |
| **執行亮點** | 三種阻力被塑造為搞怪動畫角色，嘲弄和干擾七位帕運選手，既幽默又有力 |
| **結果** | 英國史上最多人收看的帕運轉播（自 2012 年以來）|

**創意手法標籤：** `視角顛覆` `去污名化` `角色化抽象概念` `情感策略（幽默+力量）`

## 五、廣告創意手法系統框架

### 5.1 洞察（Insight）的層次

好的洞察不是市場數據，而是「消費者內心沒說出口但深有同感的真相」。

| 洞察層次 | 說明 | 案例 |
|----------|------|------|
| **行為洞察** | 消費者「做什麼」的行為真相 | KitKat：人們拿起手機是焦慮，不是休息 |
| **情緒洞察** | 消費者「感受到什麼」的情緒真相 | Speak：台灣人不敢說英文是恐懼，不是不會 |
| **文化洞察** | 社會共有的矛盾或張力 | 天天衛生棉：社會對「女生」的刻板框架 |
| **產業洞察** | 競品都在做的事，但沒人說出的潛規則 | Dove：美妝業靠不真實形象獲利 |

### 5.2 創意顛覆邏輯

得獎廣告幾乎都包含一個「意外轉折」——打破受眾的預期認知。

| 顛覆類型 | 說明 | 案例 |
|----------|------|------|
| **品牌定位顛覆** | 品牌反向主張（做不好 / 不完美 / 最爛） | Speak「英文講最爛代言人」 |
| **敘事視角顛覆** | 改變「說故事的人」或「故事的主角」| Channel 4：重力和摩擦力才是主角 |
| **產品角色顛覆** | 產品不再是功能性工具，而是文化符號或社會力量 | AXA：保險條款成為社會政策 |
| **媒介顛覆** | 廣告出現在「廣告不該在的地方」| L'Oréal：品牌影片進入 TED |
| **時間顛覆** | 極短（1 秒）或極長（紀錄片）的時間格式 | Budweiser：1 秒廣告 |

### 5.3 情感策略矩陣

| 情感類型 | 適合品牌訴求 | 風險 |
|----------|-------------|------|
| **共鳴（Resonance）** | 品牌價值觀與消費者生命經驗交疊 | 若洞察不精準，顯得虛假 |
| **幽默（Humor）** | 降低廣告排斥感，增加分享動機 | 若文化脈絡不對，顯得輕浮 |
| **驚喜（Surprise）** | 打破預期，強化記憶點 | 若只為驚喜而驚喜，品牌連結薄弱 |
| **憤慨（Righteous Anger）** | 呼應社會不公，品牌站在消費者這邊 | 品牌立場需真實，否則反噬 |
| **敬畏（Awe）** | 宏大視覺或數據衝擊，強化品牌格局感 | 易流於空洞壯觀 |

### 5.4 文化時機策略（Culture Jacking）

利用社會正在發生的話題、事件或議題，把品牌訊息嵌入文化討論。

**操作要點：**
1. **時機精準**：需在話題熱度高峰時出現，晚了效果減半
2. **品牌相關性**：品牌切入的角度必須與產品有邏輯連結（不能硬蹭）
3. **態度清晰**：不能兩面都說，品牌需有明確立場
4. **準備預案**：事件走向有時難預料，需提前準備應對版本

| 時機類型 | 說明 | 案例 |
|----------|------|------|
| **預期事件（Predictable）** | 奧運、選舉、過年等定期可規劃 | 天天衛生棉 × 巴黎奧運林郁婷 |
| **突發議題（Real-time）** | 需要快速反應，24–48 小時內上線 | 即時社群廣告 |
| **文化符碼（Evergreen）** | 長期存在的社會心理，不限時間 | Dove Real Beauty（21 年議題持續演進）|

### 5.5 品牌行動主義（Brand Activism）

2025 年越來越多得獎廣告的核心是「品牌做了什麼」，而不只是「品牌說了什麼」。

| 層次 | 說明 | 案例 |
|------|------|------|
| **說（Say）** | 品牌表態支持某種價值觀 | 最基本，常被批評為「washing」 |
| **做（Do）** | 品牌改變自身政策 / 產品 / 供應鏈 | AXA 在保險條款中加入家暴保障 |
| **推動（Catalyze）** | 品牌行動引發產業或社會的連鎖改變 | AXA 推動其他保險公司跟進 |

> **CD 的判斷標準**：如果廣告訴求是「品牌說」，問問自己：「如果競品也說同樣的話，這個廣告還有效嗎？」如果答案是「有」，說明缺乏真正的品牌差異化。

## 六、評審的思維模式（如何讓作品得獎）

### Cannes 評審的核心問題

根據歷屆評審主席發言整理：

1. **「這個洞察是真的嗎？」** — 我作為一個人，在廣告播出前有沒有感受過這個真相？
2. **「這個創意是必然的嗎？」** — 這個洞察只能引發這個創意，還是可以換成其他執行？
3. **「執行達到洞察的高度了嗎？」** — 最好的洞察配上平庸執行，不會得獎
4. **「它改變了什麼？」** — 看完廣告後，觀眾的想法、行為或世界有改變嗎？
5. **「它只能由這個品牌做嗎？」** — 把 Logo 換成競品，這個廣告還成立嗎？

### D&AD 評審標準（2025 年強調）

- **工藝（Craft）的精準度**：每個選擇都是刻意的，不是「看起來不錯」
- **敘事的完整性**：概念、執行、媒介三者是同一個故事
- **商業影響的真實性**：創意必須服務於真實的商業或社會目標

### 台灣 4A 評審的在地偏好

- **文化連結深度**：廣告是否真正理解台灣消費者的日常心理
- **媒體擴散力**：廣告在社群的自然發酵能力（分享、討論、二創）
- **整合溝通完整性**：單一影片 + 整合行銷執行的系統性
