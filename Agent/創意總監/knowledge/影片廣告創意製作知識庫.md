# 影片廣告創意製作知識庫（創意總監視角）

> 聚焦影片廣告的創意發展、製作流程把關、跨媒體影片策略。
> 資料截止日期：2025–2026 年。

---

## 一、影片廣告創意策略框架

### Hero / Hub / Help 模型（Google 提出）

| 層次 | 說明 | 製作投入 | 發布頻率 |
|------|------|----------|----------|
| **Hero（英雄）** | 大型品牌主片，追求廣泛觸達與話題性 | 最高預算，電影級製作 | 每年 1–2 次（如過年、新品上市）|
| **Hub（樞紐）** | 固定欄目型內容，維持品牌存在感 | 中等預算 | 每月 / 每季 |
| **Help（幫助）** | 回應消費者問題的實用內容 | 低預算，快速製作 | 持續發布 |

> **CD 應用洞察**：大多數品牌只做 Hero，忽略 Hub 和 Help。真正有效的影片策略需三者並行，Hero 片打品牌，Hub/Help 片做轉換。

### 影片廣告的情緒策略矩陣

| 目標 | 情緒策略 | 適合影片類型 |
|------|----------|-------------|
| **品牌認知** | 驚喜、感動、幽默 | Hero Film、品牌故事 |
| **品牌偏好** | 信任、共鳴、認同 | Brand Documentary、Testimonial |
| **產品考量** | 好奇、渴望、信服 | Product Demo、Explainer |
| **購買決策** | 緊迫感、限時優惠 | Performance Ad、Retargeting 影片 |

## 二、TVC 腳本撰寫格式與標準

### 標準腳本格式（Script Format）

TVC 腳本分為左右兩欄：

```
畫面（VIDEO）                    | 聲音（AUDIO）
---------------------------------|----------------------------------
開場：產品放置在廚房檯面         | SFX: 廚房環境音
鏡頭慢慢推近產品                 | 旁白（VO）：「每個早晨，從這裡開始。」
特寫：產品 Logo 發光             | Music: 溫暖鋼琴旋律漸入
演員打開產品，微笑               | SFX: 開啟聲
```

### 不同時長的腳本節奏配置

**15 秒 TVC（最挑戰的格式）**

| 時間區段 | 內容 |
|----------|------|
| 0–3s | 強視覺 / 問題提出（Hook）|
| 3–10s | 產品呈現 / 解決方案 |
| 10–13s | 情感收結 |
| 13–15s | Logo + Slogan |

**30 秒 TVC（最常見格式）**

| 時間區段 | 內容 |
|----------|------|
| 0–5s | 情境建立 / 引起共鳴 |
| 5–20s | 問題 → 解決 → 產品呈現 |
| 20–27s | 情感高點 / 行動呼籲 |
| 27–30s | Logo + Slogan + Pack Shot |

**60 秒 Brand Film**

| 時間區段 | 內容 |
|----------|------|
| 0–8s | 強開場，抓住注意力 |
| 8–45s | 故事主體，衝突與轉折 |
| 45–55s | 情感高峰，品牌訊息融入 |
| 55–60s | Logo + Slogan，留下記憶 |

## 三、導演提案（Director's Treatment）的審閱標準

### CD 審閱 Treatment 的關鍵問題

1. **導演是否真正理解品牌訊息？** 視覺方向是否在服務核心訊息，還是在展現自我風格？
2. **視覺參考是否符合品牌調性？** Reference 影片的色調、節奏、演員氣質是否對齊 Brief？
3. **選角方向是否精準？** 演員的氣質能否讓目標受眾自我投射？
4. **技術手段是否必要？** Drone、高速攝影等特殊設備是否有助於傳遞訊息，還是炫技？
5. **預算合理性？** 導演提案是否在預算範圍內可執行？

### 給導演的 Briefing 要點

- 提供清晰的「不能做」清單（Mandatories）
- 分享 3–5 個「感覺方向」的參考影片（Tone Reference），不是「完全照做」
- 說明目標受眾的生活場景與情感狀態（給導演設計人物用）
- 說明媒體通路：若要做 9:16 版本，導演需從一開始就規劃構圖

## 四、PPM（Pre-Production Meeting）的 CD 任務

PPM 是廣告製作中最關鍵的決策節點，Creative Director 在 PPM 中需：

### PPM 前的準備

- [ ] 再次確認 Brief 中的核心創意訊息
- [ ] 審閱分鏡（Storyboard）每一格，確認鏡頭意圖符合腳本
- [ ] 確認選角照片，評估演員是否符合品牌形象
- [ ] 確認場景 / 道具 / 服裝的品牌調性一致性
- [ ] 提前準備修改清單，PPM 是最後的修改時機

### PPM 中的 CD 職責

| 項目 | CD 的關注點 |
|------|------------|
| **腳本逐格確認** | 每個鏡頭的「訴求意圖」是否清晰 |
| **選角確認** | 演員氣質、性別年齡是否符合目標受眾 |
| **服裝道具** | 顏色是否干擾品牌色？道具是否真實可信？|
| **場景設計** | 環境是否傳遞正確的品牌訊息？|
| **音樂方向** | Temp Track 是否準確傳達情緒期望 |
| **後期時程** | 是否有足夠時間做品質把關？|

## 五、社群影片廣告的創意原則

### 「3 秒鐘法則」的設計邏輯

在 TikTok、Instagram Reels、YouTube Shorts 中，用戶決定是否繼續看的時間是 **前 3 秒**。

**強開場的 4 種策略：**

1. **視覺衝擊（Visual Punch）**：異常的畫面、強烈的色彩對比
2. **問題鉤子（Question Hook）**：「你知道為什麼...？」引發好奇
3. **情緒共鳴（Emotional Trigger）**：一個讓目標受眾立刻有感的情境
4. **品牌直球（Brand First）**：產品直接出現，針對已知品牌的再行銷廣告

### UGC（用戶生成內容）風格廣告

**為什麼 UGC 廣告在 2025 有效：**
- 與平台原生內容的視覺語言一致，降低「廣告排斥感」
- 真實感強，品牌信任度高
- 製作成本低，可快速大量測試

**CD 如何品牌化 UGC 廣告：**
- 保留真實感的拍攝方式（手持、自然光、對話語氣）
- 在後期加入品牌色的字幕設計，維持識別度
- 結尾必須有清晰的 CTA 和品牌識別
- 避免過度修飾，「太精緻」的 UGC 反而喪失效果

## 六、影片廣告的版本管理（Versioning）

一個廣告影片通常需要製作多個版本：

### 時長版本

| 版本 | 用途 |
|------|------|
| **60s（Long Form）** | YouTube、品牌官網、Hero 主片 |
| **30s（Standard）** | 電視廣告、YouTube Pre-roll |
| **15s（Short Cut）** | 電視廣告加購、Instagram、Facebook |
| **6s（Bumper）** | YouTube Bumper Ad，品牌認知用 |

### 格式版本

| 版本 | 用途 |
|------|------|
| **16:9（橫式）** | YouTube、電視、電腦螢幕 |
| **9:16（直式）** | TikTok、Reels、Stories |
| **1:1（方形）** | Facebook / Instagram Feed |
| **4:5（Portrait）** | Instagram Feed 最大面積 |

### 語言 / 地區版本

- 更換 VO（配音）/ 字幕即可的版本：**基礎本地化**
- 需要更換演員或場景的版本：**深度在地化**（成本高，需在製作前規劃）

> CD 的任務：在前期 Brief 階段就確認需要哪些版本，並讓導演在拍攝時一次到位（多比例構圖、多語言 VO 錄製）。

## 七、影片廣告效果把關

### 上線前的 CD 檢核清單

**創意層面**
- [ ] 品牌訊息在最後 3 秒仍清晰可讀？
- [ ] 在靜音播放下（字幕）廣告仍有效？
- [ ] Logo 和 CTA 在手機小螢幕可識別？
- [ ] 前 3 秒是否有足夠的視覺吸引力？

**技術層面**
- [ ] 各平台規格版本已輸出（解析度、時長、格式）？
- [ ] 字幕文件（SRT）已製作？
- [ ] 色彩確認在目標裝置（iPhone、電視）顯示正確？
- [ ] 音量標準符合各平台規格？

## 八、廣告創意效果的數據解讀

### 創意測試方法（Pre-Launch）

| 方法 | 說明 | 適合階段 |
|------|------|----------|
| **Animatic Test** | 用動態分鏡測試腳本說服力 | 腳本確認後 |
| **LINK Test（Kantar）** | 定量測試品牌連結、訴求理解、情緒反應 | 正式上線前 |
| **Concept Screening** | 多個創意方向快速篩選 | 概念發展階段 |
| **Copy Testing** | 在線上小量投放A/B測試 | 上線前低成本測試 |

### 上線後的創意指標解讀

| 指標 | 正常範圍 | CD 解讀角度 |
|------|----------|------------|
| **前 3 秒留存率** | 40–70% | 開場設計是否夠強 |
| **完播率（VTR）** | 15s：40%；30s：25% | 內容是否有持續吸引力 |
| **品牌記憶度（Ad Recall）** | 高於行業均值 10%+ | 品牌識別設計是否有效 |
| **搜尋提升（Search Lift）** | 正向 | 廣告是否引發主動興趣 |
| **情感反應（Sentiment）** | 正面留言 > 負面 | 創意是否引起共鳴或反感 |

## 九、影片廣告製作的常見錯誤

| 錯誤類型 | 說明 | 如何避免 |
|----------|------|----------|
| **品牌太晚出現** | Logo 在 25 秒才出現，流失前跳過的觀眾 | 前 5 秒帶入品牌視覺線索 |
| **訊息過多** | 30s 塞入 5 個產品功能 | 每支影片只講一件事 |
| **靜音無效** | 廣告完全依賴語音傳遞訊息 | 視覺和文字同步傳達核心訊息 |
| **格式忽略** | 只剪橫式，後製裁切 9:16 損失關鍵畫面 | 前期規劃多比例拍攝構圖 |
| **結尾太弱** | 廣告情緒高點後 Logo 突然冷場出現 | 設計情感延續到品牌識別的過渡 |
| **過度依賴 AI 生成** | AI 影像品質不穩定，品牌感低 | AI 用於 Previs，最終執行仍需製作 |

## 十、企業形象影片（Brand Film）提案腳本架構

> 本節知識源自藍麒科技_貿協提案用腳本的實務分析，適用於 2–3 分鐘的企業品牌影片提案。

### 核心訊息（Core Message）的撰寫框架

核心訊息是整支影片的靈魂，須包含三個層次：

| 層次 | 說明 | 藍麒科技範例 |
|------|------|-------------|
| **主題框架** | 以一個具象概念包裝品牌精神 | 「The Crucible（實驗箱）」——極端測試即是品牌最好的證明 |
| **一句標語** | 精煉的品牌主張 | 《極致恆溫的最高標準》 |
| **說明段落** | 50 字內說明影片核心概念與主視覺方向 | 強調「12米一體成型技術」作為消除接縫、解決溫控的終極方案 |

**CD 工作原則：** 核心訊息不是功能清單，而是一個「讓品牌有理由存在」的故事框架。找到一個足以統攝所有競爭優勢的敘事核心。

---

### 7 段企業品牌影片架構

企業形象影片（約 2 分半）的標準七段結構：

| 段落 | 功能 | 時間碼 | 建議字卡風格 |
|------|------|--------|-------------|
| **1. 核心開場** | 情感鉤子，建立品牌第一印象 | 0:00–0:20 | 年份/歷史數字，強調積累 |
| **2. 競爭力 A** | 第一組差異化陳述（通常是材料/原料優勢）| 0:20–0:40 | 具體合作品牌/認證名稱 |
| **3. 競爭力 B** | 關鍵競爭力強化（技術面/實測面）| 0:40–1:00 | 無字卡，讓畫面與VO說話 |
| **4. 領先工藝** | 核心技術的視覺落地 | 1:00–1:25 | 技術規格數字（如「12米」） |
| **5. 整合實力** | 全方位服務能力，建立信任感 | 1:25–1:40 | 無字卡，品管/流程畫面為主 |
| **6. 願景總結** | 升華品牌高度，連結市場願景與 ESG | 1:40–1:55 | 認證標準、客戶產業名稱 |
| **7. 品牌精神** | Slogan + LOGO，情感收尾 | 1:55–2:00 | Slogan + LOGO |

**架構邏輯：** 開場建立信任基礎（歷史+規模）→ 中段展開技術差異化 → 後段升華至市場願景 → 結尾情感收斂至品牌。

---

### VO 旁白寫作原則（企業品牌影片）

從藍麒科技腳本萃取的旁白風格標準：

**語氣：** 沉穩、有力、略帶詩意，不過度煽情，不使用廣告式直白語言

**結構：**
- 每段旁白控制在 40–60 字，讀出來約 15–20 秒
- 多用短句與並列句，避免長句堆疊
- 開場句通常是品牌宣言或具象場景描述
- 結尾句留有餘韻，避免直白的廣告感

**範例對照：**
- ✅ 「一片庫板，凝聚了20年的研發與遠見。」（強宣言開場）
- ✅ 「我們阻絕熱能的流動，也守護空間內對溫控的精準。」（技術+詩意）
- ✅ 「冷鏈是食品安全、醫療品質與半導體製程的根基。」（市場願景升華）
- ❌ 「我們的產品品質非常好，請大家多多支持。」（過於直白）
