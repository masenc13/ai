## 一、演算法行銷概論：CD 為何必須懂演算法

過去的創意總監只需要懂「說什麼」和「怎麼說」。
現在還需要懂「演算法決定讓誰看到」。

**核心邏輯：**
> 創意再好，若演算法不分發，等於沒人看到。
> 演算法偏好的內容，不一定是創意最好的內容，但一定是**行為訊號最強**的內容。

**CD 的角色轉變：**

| 過去 | 現在 |
|------|------|
| 為媒體購買時間設計創意 | 為演算法設計觸發訊號 |
| 30s TVC 是核心格式 | 依平台演算法偏好格式設計 |
| 大預算＝大觸及 | 演算法判斷觸及，預算只是放大器 |
| 品牌說什麼決定效果 | 用戶行為反應決定演算法分發 |

## 二、演算法行銷的主要類型

| 類型 | 說明 | CD 的應用場景 |
|------|------|--------------|
| **社群媒體演算法** | 決定有機內容的分發範圍 | 社群貼文、Reels、TikTok 影片策略 |
| **程序化廣告演算法** | 即時競標、精準投放付費廣告 | 廣告素材多版本、動態創意優化 |
| **搜尋演算法（SEO）** | 決定品牌在 Google / TikTok Search 的排名 | 品牌內容、官網、KOL 文章 |
| **推薦引擎** | Netflix / Spotify 等平台的內容推薦 | Branded Content 的平台合作策略 |
| **電商演算法** | Shopee / MOMO / Amazon 的搜尋與推薦排序 | 品牌電商頁面設計、促銷時機 |
| **廣告投放演算法（ML Bidding）** | Google / Meta 自動出價優化 | 廣告素材測試策略、A/B 設計 |

## 三、TikTok FYP 演算法深度解析

### 運作邏輯

TikTok 是目前**對品牌最民主也最殘酷**的演算法：帳號追蹤數不影響初始分發，內容品質決定一切。

**分發漏斗（Distribution Funnel）：**

```
第一輪投放：500–1,000 人的種子受眾（依內容標籤、聲音、興趣匹配）
         ↓ 若完播率 > 50%
第二輪放大：1 萬–5 萬人
         ↓ 若持續高完播率 + 互動
第三輪病毒：百萬以上曝光（FYP 大規模分發）
```

### 核心排名訊號（依重要性排序）

| 訊號 | 說明 | 行業基準 |
|------|------|----------|
| **完播率（Completion Rate）** | 看完整段影片的比例——最關鍵指標 | 50%+ 開始大量分發；30% 以下幾乎停止 |
| **重播率（Replay Rate）** | 用戶主動重播的比例 | 代表內容有強力記憶點或娛樂性 |
| **分享（Shares）** | 轉發到聊天室或外部平台 | 高分享 = 強情感共鳴或實用價值 |
| **留言（Comments）** | 留言數量與情緒 | 爭議性內容易引發高留言，但要謹慎 |
| **按讚（Likes）** | 基本訊號，權重相對較低 | 完播率和分享更重要 |
| **關注（Follows）** | 看完後點關注 | 代表受眾認同創作者定位 |

### TikTok Search 的崛起（2025 關鍵趨勢）

2025 年，**TikTok 正式超越 Google，成為 Z 世代的首選搜尋工具**。

- 用戶在 TikTok 上搜尋：產品評測、旅遊建議、料理食譜、生活技巧
- TikTok 演算法開始優先分發**能回答搜尋問題的內容**
- 品牌機會：製作「#什麼是X」「怎麼選X」「X的評測」系列，同時優化 TikTok SEO

**TikTok SEO 優化要點：**
- 影片標題含關鍵字（前 3 行可見區域）
- 字幕中含關鍵字（TikTok OCR 掃描字幕文字）
- hashtag 使用大分類 + 小眾精準標籤組合
- 影片前 3 秒口述關鍵字（TikTok 語音識別）

### 品牌在 TikTok 的創意原則

1. **前 1–2 秒是生死線**：用戶條件反射式滑動，開場必須立即抓住注意力
2. **拒絕廣告感**：TikTok 用戶對「明顯廣告」的耐受度接近零，原生感是必要條件
3. **完播設計優先**：結尾要有「回饋」——問題的解答、驚喜轉折、笑點
4. **聲音是必要元素**：TikTok 是聲音平台，熱門音樂和獨特音效本身就是演算法訊號
5. **鼓勵互動的句式**：「你是哪一種？」「留言告訴我」「雙擊如果你也...」

## 四、Instagram 多重演算法深度解析

### 2025–2026 年重大改變

Instagram 官方宣布放棄「單一演算法」，改為**針對不同版面的獨立 AI 排名系統**：

| 版面 | 演算法目標 | 主要用戶行為 |
|------|------------|------------|
| **Feed** | 展示你關心的帳號的最新內容 | 找朋友和感興趣帳號的貼文 |
| **Stories** | 展示最親密關係的即時動態 | 追蹤最親近好友 |
| **Reels** | 發現新內容、娛樂 | 探索不認識的創作者 |
| **Explore** | 發現符合興趣的新帳號 | 主動探索 |
| **Search** | 回應明確的關鍵字意圖 | 搜尋特定主題或帳號 |

### Feed / Stories 排名訊號

- **關係親密度**：DM 往來頻率、互相標記、留言互動歷史
- **興趣比對**：過去對同類內容的互動行為
- **發布時效性**：越新越好（Stories 尤其重視）

### Reels 排名訊號（2025–2026 確認版本）

Adam Mosseri（Instagram CEO）2025 年 1 月確認前三大訊號：

| 排名 | 訊號 | 說明 |
|------|------|------|
| 1 | **觀看時長（Watch Time）** | 看完 Reels 的比例，前 3 秒是否留住是關鍵門檻 |
| 2 | **按讚（Likes）per Reach** | 每次觸及所產生的按讚比率（代表內容品質密度）|
| 3 | **私訊分享（DM Shares）** | 傳給朋友的次數——最高品質的分享行為 |

**2026 年重要更新：**
- Reels 延長至最長 **20 分鐘**：長形式教學、幕後紀錄、迷你紀錄片開始被 Explore 推薦
- **原創內容加分、轉發扣分**：30 天內 10 次以上轉發他人內容的帳號被排除推薦；原創帳號觸及率提升 40–60%
- Instagram 貼文開始被 **Google 收錄**：公開的專業帳號貼文可出現在 Google 搜尋結果

### 品牌在 Instagram 的創意原則

1. **Reels 是主要觸及引擎**：靜態貼文觸及率下滑，Reels 觸及率 2.46%（2025 Sprout Social 基準）
2. **DM 誘發設計**：創意中加入「傳給最需要的朋友」類呼籲，直接驅動最高權重訊號
3. **原創內容是唯一出路**：演算法對轉發帳號的懲罰已十分明確
4. **Collab Post（合作貼文）**：與 KOL 共同發佈，一篇貼文同時觸達兩個帳號的粉絲
5. **Stories 維持關係、Reels 擴大觸及**：兩個版面應有不同的內容策略

## 五、YouTube 演算法深度解析

### 2025 年重大演算法更新：滿意度加權發現

YouTube 2025 年初宣布推薦模型大幅調整，從**觀看時長優先**升級為**「滿意度加權發現（Satisfaction-Weighted Discovery）」**：

**新系統的組成：**

| 訊號層次 | 說明 | 新舊差異 |
|----------|------|----------|
| **滿意度調查** | 觀影後彈出「你喜歡這支影片嗎？」 | **新**：首度成為核心排名因素 |
| **情緒建模** | 分析留言情緒、按讚 / 倒讚比例 | 強化，不再只看數量 |
| **跨影片留存** | 看完後繼續在平台觀看的時長 | 強化（衡量整體滿意度）|
| **「不感興趣」點擊** | 用戶主動排除某頻道或類型 | 懲罰力度提升 |
| **觀看時長（Watch Time）** | 仍重要，但不再是唯一核心 | 降權（從第一降至第三）|

### YouTube 的核心排名訊號對照

| 訊號 | 目標值 | 說明 |
|------|--------|------|
| **點擊率（CTR）** | 5–7% 為健康；10%+ 為優秀 | 縮圖 + 標題決定是否點擊 |
| **觀眾留存率（Audience Retention）** | 50% 中段留存為基準線 | 影片中段離開比例反映內容節奏問題 |
| **觀看完成率** | 60%+ 為佳 | 短影片完播更容易，長影片要在結構上引導 |
| **滿意度** | 以調查結果衡量 | 2025 年後成為最重要的隱性訊號 |

### YouTube 分發漏斗

```
發布後 24 小時：給予約 1,000 次曝光測試 CTR
  ├─ CTR ≥ 10%（100 次點擊）→ 大幅擴增曝光（指數成長）
  ├─ CTR 5–9% → 緩慢擴增，觀察後續留存率
  └─ CTR < 4%（40 次點擊）→ 幾乎停止推薦
```

### 品牌在 YouTube 的創意原則

1. **縮圖（Thumbnail）是廣告**：品牌影片的縮圖設計需達到 5%+ CTR，用大字、強烈表情、對比色
2. **前 30 秒決定留存率**：快速建立期待感，說明「看完這支影片你會得到什麼」
3. **YouTube Shorts 與長影片連動**：Shorts 作為引流，引導訂閱頻道後看長影片
4. **品牌頻道定位明確**：演算法偏好有清晰主題定位的頻道（不是什麼都發）
5. **滿意度設計**：讓觀眾看完後「真的有收穫」，才能通過新的滿意度訊號門檻

## 六、Facebook 演算法深度解析

### 2025 年 Facebook 演算法的核心轉變

Facebook 演算法在 2025 年強化**「私訊分享」作為最高品質訊號**的判斷方式。

**演算法偏好的內容行為（依優先級）：**

| 行為 | 訊號強度 | 說明 |
|------|----------|------|
| **私訊分享（Share via Messenger）** | 最高 | 用戶主動傳給特定朋友，最真實的內容認可 |
| **留言討論** | 高 | 有實質討論內容的留言（非表情符號）|
| **按讚 / 互動** | 中 | 基本訊號，易被刷出，演算法已降低權重 |
| **觀看影片時長** | 中高（影片內容）| 影片觀看 1 分鐘以上開始加分 |
| **公開分享** | 中 | 分享到個人塗鴉牆 |

### 品牌在 Facebook 的創意方向

- **設計「值得私訊分享」的內容**：實用資訊（不知道你不行的知識）、情感共鳴（說出了別人的心聲）、幽默（讓你忍不住傳給朋友）
- **Facebook 社團（Groups）**：演算法大力推薦社團內容，品牌可建立官方社團或贊助相關社團
- **Live 直播**：Facebook 對 Live 給予更高自然觸及，品牌活動直播是有效低成本觸及方式

## 七、程序化廣告（Programmatic Advertising）演算法

### 生態系結構

```
廣告主（Advertiser）
    ↓ 透過
DSP（Demand-Side Platform）需求方平台
    ↓ 連接
Ad Exchange 廣告交易所（即時競標）
    ↑ 連接
SSP（Supply-Side Platform）供應方平台
    ↑ 提供廣告版位
出版商 / 媒體（Publisher）
```

**即時競標（RTB）運作時間：約 100 毫秒完成整個競標→投放流程**

### 程序化廣告的演算法類型

| 類型 | 說明 | CD 的意義 |
|------|------|----------|
| **受眾定向演算法** | 依人口統計、興趣、行為、再行銷比對目標受眾 | 決定「誰看到廣告」 |
| **出價優化演算法** | 自動調整出價以最大化轉換或曝光效益 | 影響廣告素材的展示頻率 |
| **動態創意優化（DCO）** | 依受眾特徵自動組合廣告素材元素 | 素材設計需模組化，讓演算法有組合空間 |
| **情境比對演算法** | 依頁面內容相關性投放廣告（不依賴 Cookie）| Cookie 消亡後的替代方案 |
| **頻率控制演算法** | 控制同一用戶看到同一廣告的次數 | 避免廣告疲乏 |

### CTV（聯網電視）廣告演算法

**市場規模：** 全球 CTV 廣告市場（2025）= **USD 315 億**，2027 年預測達 USD 426 億（CAGR 16.3%）

| 特性 | 說明 |
|------|------|
| **家庭層級定向** | 以家庭 IP 為單位，精準至家庭人口組成 |
| **跨螢幕追蹤** | 連結同一家庭的手機、平板、電視行為 |
| **不可跳過** | CTV 廣告完播率高達 **90%+**（vs YouTube 可跳過廣告的 35–55%）|
| **品牌安全** | 廣告出現在正規 OTT 平台，品牌安全度高 |

**CTV 廣告的創意策略：**
- 廣告無法跳過 → 製作真正值得看完的 30s / 60s 廣告，而非「強迫觀看型」
- 第一秒就要有品牌識別（Logo 或 Slogan），因為無法依賴 5 秒後的留存
- 與串流內容的調性匹配（高端平台→高端製作質感）

### DOOH（數位戶外廣告）演算法

**市場現況：** 美國 DOOH 佔 OOH 總支出 **34%**，2025 Q1 創歷史新高 USD 19.8 億

| 特性 | 說明 |
|------|------|
| **受眾流動定向** | 依地點、時段、天氣、人流數據決定投放 |
| **即時數據觸發** | 股市指數、天氣、體育賽事比分可觸發廣告切換 |
| **動態創意** | 同一個廣告版位依時段自動展示不同版本 |
| **測量進化** | 手機位置數據追蹤「看到廣告→進入門市」的轉換 |

## 八、動態創意優化（DCO）— 演算法時代的素材策略

### 什麼是 DCO

DCO（Dynamic Creative Optimization）是由演算法**即時組合廣告素材元素**，向不同受眾展示最適合的廣告版本。

**DCO 素材組成範例：**

```
背景圖片（3 個版本）× 標題文案（4 個版本）× 產品圖（5 個版本）× CTA（3 個版本）
= 180 個廣告組合，演算法選出對每個受眾最有效的版本
```

### CD 的 DCO 設計原則

1. **模組化設計思維**：不設計「一支廣告」，設計「素材元素庫」
2. **品牌一致性框架**：不論演算法如何組合，品牌色彩、字體、視覺語言需維持一致
3. **核心訊息不可被切割**：關鍵的品牌主張必須在任何組合中都能獨立成立
4. **A/B 測試設計**：每次發布至少 3–5 個素材版本，讓演算法有優化空間
5. **避免「完美主義」**：DCO 環境下，測試比精雕細琢更重要

### Meta Advantage+（前 Dynamic Creative）

Meta 的 AI 廣告系統 Advantage+（2025）：
- 自動找到最佳受眾（不需手動設定受眾）
- 自動測試素材組合
- 自動分配預算到表現最佳的素材

**品牌使用 Advantage+ 的創意輸入建議：**

| 元素 | 建議數量 | 說明 |
|------|----------|------|
| 影片素材 | 3–6 支 | 不同開場方式、情緒切角 |
| 靜態圖 | 3–5 張 | 不同視覺構圖方向 |
| 標題文案 | 5 個 | 不同訴求重點 |
| 描述文字 | 3 個 | 功能性 / 情感性 / 促銷性各一 |

## 九、Google 搜尋演算法（SEO）— 品牌內容的長尾影響力

### E-E-A-T 框架（2025 年最重要的 SEO 評分標準）

Google 從 E-A-T 升級為 **E-E-A-T（Experience, Expertise, Authoritativeness, Trustworthiness）**：

| 維度 | 說明 | 品牌的對應策略 |
|------|------|--------------|
| **Experience（經驗）** | 內容創作者是否有親身體驗 | 真實用戶故事、品牌紀錄、第一人稱敘事 |
| **Expertise（專業）** | 對主題的專業深度 | 品牌官方教學、產業白皮書、深度說明文章 |
| **Authoritativeness（權威）** | 在特定領域被引用、被信任的程度 | 媒體報導、KOL 引用、外部連結 |
| **Trustworthiness（信任）** | 資訊準確性、透明度、安全感 | 清楚的來源標示、沒有誤導性訊息 |

### 2025 年 Google 搜尋的新現實

- **AI 概覽（AI Overviews）** 出現在大量搜尋結果頂部，擠壓傳統前 3 名的點擊率
- **品牌官網不再是唯一陣地**：YouTube、Reddit、TikTok 上的品牌相關內容也影響 SEO
- **TikTok 貼文開始被 Google 收錄**：社群影片出現在 Google 搜尋結果
- **「有用性（Helpfulness）」是核心標準**：2024–2025 年 Core Update 大量懲罰低品質、AI 生成的薄弱內容

### CD 的 SEO 思維框架

> SEO 不只是技術部門的事。品牌的每一篇深度內容、每一支教學影片、每一個 KOL 合作文章，都在建構搜尋能見度。

**品牌 SEO 內容策略方向：**
- **Pillar Content**（支柱內容）：深度長文，建立品牌在核心主題的權威地位
- **FAQ 型影片**：針對消費者常搜尋的問題製作影片，同時優化 YouTube 和 Google
- **KOL 內容 × SEO**：請 KOL 在部落格 / YouTube 撰寫含品牌關鍵字的深度測評

## 十、自然觸及 vs 付費放大 — 演算法時代的預算策略

### 2025 年自然觸及的真實現狀

| 平台 | 品牌帳號自然觸及率（2025）| 趨勢 |
|------|--------------------------|------|
| Facebook 品牌頁面 | **1–2%**（貼文觸及追蹤者比例）| 持續下滑 |
| Instagram 品牌帳號 | **2–3%**（靜態貼文）/ **4–6%**（Reels）| Reels 相對較高 |
| TikTok | **0–300%+** | 高度依賴演算法判斷，極端兩極化 |
| YouTube | 無固定基準，依演算法信號差異極大 | 訂閱者觸及率約 10–30% |

### 最有效的混合策略（2025 年 CMO 共識）

**「聽→放大」模型（Listen-Then-Amplify）：**

```
發布有機內容（Organic Post）
    ↓ 觀察 48–72 小時
    ├─ 互動率超過閾值（如 3%+）→ 付費放大（Boost）
    └─ 表現低於預期 → 分析原因，調整下一個有機內容
```

**預算配置建議（總活動預算）：**

| 類型 | 預算佔比 |
|------|----------|
| 有機內容製作 | 30–40% |
| 付費放大（優秀有機內容的 Boost）| 30–40% |
| 純付費廣告投放（Conversion Campaign）| 20–30% |

### Creator Whitelisting（創作者白名單廣告）

**定義**：從 KOL / 創作者的帳號投放付費廣告，而非品牌官方帳號。

**為何有效：**
- 廣告外觀像有機內容 → 演算法和用戶都更接受
- 利用創作者的「真實感」加持品牌訊息
- 能鎖定創作者的粉絲受眾，且可再擴展至相似受眾（Lookalike）

**執行要點：**
1. 在創作者合約中包含白名單廣告授權條款
2. 廣告版本與創作者自然風格保持一致（勿過度品牌化）
3. 從創作者帳號投放時，監測的 KPI 應與品牌帳號廣告分開計算

## 十一、演算法時代的創意設計原則（CD 應用框架）

### 各平台「演算法友好」創意設計要點

| 平台 | 最關鍵訊號 | 創意設計核心原則 |
|------|------------|----------------|
| **TikTok** | 完播率 | 結尾必須有「回饋」；前 2 秒強鉤子；拒絕廣告感 |
| **Instagram Reels** | 觀看時長 + DM 分享 | 設計「值得傳給朋友」的內容；前 3 秒視覺強衝擊 |
| **YouTube** | 滿意度 + CTR | 縮圖設計如廣告；內容必須真的有用或娛樂性 |
| **Facebook** | 私訊分享 + 留言深度 | 觸發情感分享（笑、哭、憤慨）；問題型貼文引發討論 |
| **Google SEO** | E-E-A-T + Helpfulness | 深度有用，非薄弱 AI 內容；品牌真實經驗 |
| **CTV / DOOH** | 完播率 + 品牌記憶度 | 第一秒品牌識別；無法跳過，品質要值得看完 |

### 演算法時代的創意悖論

**演算法喜歡的** vs **創意人擅長的** 之間的張力：

| 演算法偏好 | 創意風險 |
|-----------|----------|
| 快速、高能量的開場 | 壓縮了故事的醞釀空間 |
| 清晰的 CTA 和互動呼籲 | 降低了廣告的藝術感 |
| 真實感、低製作感（UGC 風格）| 與品牌「高品質製作」形象衝突 |
| 頻繁發布（演算法喜歡活躍帳號）| 稀釋創意品質 |

**CD 的解法：**
> 「演算法是分發機制，不是創意標準。」
> 讓演算法訊號設計**包覆**在創意之內，而不是讓演算法邏輯**替換**創意思維。
> 例：用有力的開場鉤子服務完播率，但鉤子本身必須是真實的品牌故事或洞察。

## 十二、演算法行銷的衡量框架

### 各演算法類型的核心 KPI

| 演算法類型 | 主要 KPI | 次要 KPI |
|------------|---------|---------|
| **TikTok FYP** | 完播率、分享數、FYP 觸及 | 留言數、跟進搜尋量 |
| **Instagram Reels** | 觀看時長、DM 分享、非粉絲觸及 | 按讚率、儲存數（Saved）|
| **YouTube** | CTR、觀眾留存率、滿意度訊號 | 訂閱轉換率、跨影片留存 |
| **程序化廣告** | ROAS、CPM、VCR（完播率）| CTR、轉換率、頻率 |
| **CTV** | VCR（完播率）、品牌記憶提升 | 跨螢幕轉換（Store Visit）|
| **Google SEO** | 自然搜尋流量、排名關鍵字數 | CTR、停留時間、跳出率 |

### 演算法行銷的整合歸因挑戰

消費者在多個演算法環境中接觸到品牌，單一平台的歸因往往失真：

**建議採用的測量方式：**
- **行銷組合模型（MMM）**：統計整體行銷投資與銷售結果的關聯
- **增量測試（Incrementality Test）**：關掉某個渠道的投放，測量銷售是否實際下滑
- **品牌健康追蹤（Brand Health Tracking）**：定期調查知名度、考慮度、偏好度的長期趨勢
