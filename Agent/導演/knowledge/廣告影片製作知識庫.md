# 廣告影片製作知識庫（導演視角）

> 資料截止日期：2025–2026 年。聚焦 TVC、品牌影片、數位廣告影片的實務製作知識。

---

## 一、廣告影片類型定義

| 類型 | 時長 | 主要用途 | 製作規模 |
|------|------|----------|----------|
| **TVC（Television Commercial）** | 15s / 30s / 60s | 電視、OTT 串流廣告 | 大型製作，有完整製作團隊 |
| **Hero Film（英雄影片）** | 60s–3min | 品牌大片、YouTube 主力影片 | 大型，電影級製作規格 |
| **Brand Documentary** | 3–10min | 品牌故事、社會責任 | 中型，紀實風格 |
| **Product Demo / Explainer** | 30s–2min | 產品介紹、功能展示 | 中小型，清晰傳遞資訊 |
| **Social Film（社群影片）** | 6s–60s | Instagram / TikTok / YouTube Shorts | 小型，敏捷製作 |
| **Influencer Brief Film** | 30s–90s | KOL 合作，帶入品牌訊息 | 輕量，偏原生風格 |

## 二、廣告影片製作三大階段

### 前期製作（Pre-Production）

| 步驟 | 說明 |
|------|------|
| **創意 Brief 解讀** | 深讀 Creative Brief，釐清核心訊息、目標受眾、品牌調性 |
| **導演提案（Director's Treatment）** | 提交視覺方向、參考影片（Reference）、拍攝手法、選角方向 |
| **PPM（Pre-Production Meeting）** | 與客戶、創意總監、製片確認所有製作細節（最關鍵的溝通節點） |
| **選角（Casting）** | 真人演員或 UGC 風格；需符合品牌形象 |
| **場景勘景（Location Scouting）** | 實景 vs 棚拍；燈光條件評估 |
| **分鏡腳本（Storyboard）** | 每個鏡頭視覺化，含攝影機運動標示 |
| **Animatic（動態分鏡）** | 用靜態圖加時間軸呈現節奏，讓客戶預視成片感 |
| **拍攝計畫（Shooting Schedule）** | 每個場景的拍攝順序、時間預估、所需資源 |

### PPM 的關鍵議程
1. 腳本逐格確認（鏡頭意圖、對話、動作）
2. 選角結果確認
3. 場景 / 佈景設計確認
4. 服裝 / 道具（Wardrobe & Props）
5. 音樂方向與授權方式
6. 後期製作時程（剪接、配音、上色、特效）
7. 交付格式清單

### 拍攝日（Production Day）

| 職位 | 廣告拍攝中的角色 |
|------|----------------|
| **導演** | 決策所有創意執行，與 DOP 即時溝通調整 |
| **DOP（攝影指導）** | 掌管攝影機、燈光、曝光，執行導演視覺意圖 |
| **製片（Producer）** | 控管時程、預算、現場後勤 |
| **廣告公司代表（Agency Rep）** | 現場把關品牌訊息與腳本準確性 |
| **客戶代表（Client）** | 最終核准畫面方向 |

> 廣告拍攝的拍攝日通常 1–3 天（TVC），需要比電影更快速的決策節奏。

### 後期製作（Post-Production）

| 流程 | 說明 |
|------|------|
| **粗剪（Rough Cut）** | 剪接師依腳本順序剪輯，展示敘事邏輯 |
| **精剪（Fine Cut）** | 加入音樂、音效、字幕，精調節奏 |
| **Online Edit** | 最終剪接版本，確認無誤 |
| **色彩調校（Color Grading）** | 建立品牌色調，通常在 DaVinci Resolve 操作 |
| **VFX / 合成** | 特效、產品合成、場景延伸（如需要） |
| **配音（VO）** | 旁白錄製、音效設計（Sound Design）、音樂混音 |
| **字幕版本** | 無聲播放需求（社群平台佔多數用戶關閉聲音） |
| **最終交付（Delivery）** | 依媒體平台規格輸出不同版本 |

## 三、導演提案（Director's Treatment）結構

1. **對腳本的理解（Script Interpretation）**：導演如何解讀品牌訊息與情感
2. **視覺方向（Visual Approach）**：色調、攝影風格、燈光方向
3. **參考影片（Reference）**：3–5 部影片或導演自身作品
4. **選角方向（Casting Direction）**：演員氣質描述
5. **音樂方向（Music Direction）**：節奏感、風格關鍵字
6. **技術說明（Technical Notes）**：攝影機、鏡頭、特殊設備（Drone、穩定器等）

## 四、廣告影片的鏡頭語言特性

### 廣告 vs 電影的鏡頭節奏差異

| 維度 | 電影 | TVC（30s）|
|------|------|-----------|
| 平均鏡頭長度 | 3–6 秒 | 1.5–3 秒 |
| 資訊密度 | 分散於整部影片 | 高度濃縮 |
| 開場節奏 | 可較慢建置 | 前 3 秒必須抓住注意力 |
| 情感層次 | 多層遞進 | 單一強烈情緒衝擊 |

### 廣告常用視覺技巧

- **產品英雄鏡頭（Hero Shot）**：以最佳角度、燈光呈現產品，通常是廣告最重要的鏡頭
- **氛圍建立鏡頭（Atmosphere Shot）**：建立場景情境，引發情感共鳴
- **行為示範鏡頭（Demonstration Shot）**：展示產品功能或使用場景
- **情感收尾（Emotional Payoff）**：品牌 Logo 出現前的情感高點

## 五、廣告影片交付規格

### 電視 / OTT 廣告

| 規格項目 | 標準 |
|----------|------|
| 解析度 | 1920×1080（HD）/ 3840×2160（4K） |
| 畫面比例 | 16:9 |
| 幀率 | 25fps（台灣 / 歐洲）/ 23.976fps（電影感） |
| 音量標準 | -23 LUFS（EBU R128，歐洲）/ -24 LUFS（美國廣播） |
| 色彩空間 | Rec. 709（HD）/ Rec. 2020（HDR） |
| 格式 | ProRes 4444、ProRes 422 HQ（交付母帶）|

### 社群平台廣告規格

| 平台 | 最佳比例 | 最大時長 | 推薦解析度 |
|------|----------|----------|-----------|
| **Instagram Feed** | 4:5 或 1:1 | 60s | 1080×1350px |
| **Instagram / TikTok Stories & Reels** | 9:16 | 15–90s | 1080×1920px |
| **YouTube（In-Stream）** | 16:9 | 不限（可跳過 5s 後） | 1920×1080px |
| **Facebook Feed** | 4:5 | 240min（建議 15s–60s） | 1080×1350px |
| **YouTube Shorts** | 9:16 | 60s | 1080×1920px |

> 實務提醒：廣告拍攝時建議同時規劃 16:9、9:16 的構圖，避免後製裁切損失重要畫面。

## 六、廣告製作公司（Production House）生態

### 台灣 / 亞洲市場主要類型

| 類型 | 說明 | 代表特色 |
|------|------|----------|
| **全服務製作公司** | 完整製作能力：導演、攝影、後期一條龍 | 承接高預算 TVC |
| **精品製作公司** | 專注特定風格（如時尚、美食、科技）| 風格鮮明，客戶選導演為主 |
| **數位製作公司** | 專注社群影片、速度快、CP值高 | 敏捷製作，多版本輸出 |
| **動態設計公司（Motion Studio）** | 專注動畫、Motion Graphics | 品牌識別動態化 |

### 製作公司選擇考量

1. **導演的 Reel**：過往作品是否符合本次品牌調性
2. **製作規模**：設備、人員、後期能力
3. **溝通效率**：能否快速回應修改需求
4. **預算彈性**：是否能在預算內保持品質

## 七、廣告影片的音樂與聲音設計

### 音樂類型選擇

| 類型 | 適用情境 | 授權方式 |
|------|----------|----------|
| **Original Composition（原創配樂）** | 品牌主片、追求獨特性 | 委託作曲，費用最高 |
| **Music Library（版稅音樂庫）** | 中低預算，快速交稿 | 按次授權或年費制 |
| **Sync License（版權歌曲授權）** | 名曲帶動情緒、話題性 | 費用極高，談判複雜 |
| **Jingle（品牌歌曲）** | 電視廣告記憶點 | 委託製作，品牌資產 |

### 廣告的音效設計原則

- **產品音效（Product Sound）**：汽車引擎聲、食品開箱聲，增強感官說服力
- **情緒音效（Foley）**：環境聲、動作聲強化畫面真實感
- **無聲策略（Silent Opening）**：社群影片前 3 秒無聲抓取注意力，依賴強視覺

## 八、廣告影片獎項與評審標準

### 主要國際廣告獎項

| 獎項 | 聚焦 | 影片類別 |
|------|------|----------|
| **Cannes Lions（坎城廣告節）** | 全球最高榮譽，Film / Film Craft 類 | 創意概念 + 製作水準 |
| **D&AD（英國設計與藝術指導）** | 影視工藝：攝影、剪接、音樂 | 製作工藝為主 |
| **Clio Awards** | 創意與執行並重 | 品牌影片、數位影片 |
| **One Show** | 廣告創意，Film 類 | 創意力 |
| **時報廣告金像獎** | 台灣本地廣告殊榮 | 本地市場 |
| **亞太廣告節（ADFEST）** | 亞太區導演與創意 | 區域市場影響力 |

### 評審標準框架

1. **創意洞察（Insight）**：是否精準捕捉真實的人性洞察
2. **概念獨特性（Originality）**：在同類廣告中是否具差異化
3. **執行品質（Craft）**：攝影、剪接、音樂、表演的整體水準
4. **品牌連結（Brand Connection）**：廣告是否能強化品牌記憶
5. **文化相關性（Cultural Relevance）**：對目標受眾的文化共鳴

## 九、AI 工具在廣告影片製作的應用（2025）

| 應用場景 | 工具 | 說明 |
|----------|------|------|
| **概念視覺化** | Midjourney、DALL-E 3 | 快速生成情境圖，用於客戶提案 |
| **Animatic 製作** | Runway Gen-3、Sora | 將分鏡圖轉為動態預視 |
| **AI 粗剪** | Adobe Premiere AI、DaVinci AI | 自動組裝鏡頭，節省剪接時間 |
| **AI 配音** | ElevenLabs | 快速生成多語言 VO 版本 |
| **AI 色調調整** | DaVinci Resolve AI | 自動匹配色調，加速調色流程 |
| **VFX 輔助** | Topaz Video AI | 影片放大、去雜訊、幀率提升 |

> 行業觀察：AI 加速前期（Previs）和後期（Post）效率，但核心創意決策與現場拍攝仍需人類主導。

## 十、廣告導演的職涯路徑

| 階段 | 說明 |
|------|------|
| **導演助理 / 現場助理** | 學習拍攝流程，建立製作人脈 |
| **副導演（AD）** | 協助現場排程、管理演員走位 |
| **獨立短片 / 音樂 MV 導演** | 建立個人風格與 Reel |
| **製作公司簽約導演** | 承接商業廣告，有穩定案源 |
| **自由導演** | 個人品牌強，可跨製作公司接案 |
| **知名廣告導演** | 憑作品被大型品牌主動尋求，決定製作方向 |
