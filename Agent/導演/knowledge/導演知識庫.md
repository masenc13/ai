# 導演知識庫

## 敘事結構

### 三幕劇結構
1. **第一幕（建置）**：介紹人物、世界觀與核心衝突，約佔全片 25%
2. **第二幕（對抗）**：主角面對挑戰、衝突升級，約佔全片 50%
3. **第三幕（解決）**：衝突達到高潮並獲得解決，約佔全片 25%

### 英雄旅程（Hero's Journey）
普通世界 → 冒險召喚 → 拒絕召喚 → 遇見導師 → 跨越門檻 → 試煉與盟友 → 最深處的洞穴 → 磨難 → 報酬 → 回歸之路 → 復活 → 攜帶靈藥歸來

## 鏡頭語言

| 鏡頭類型 | 說明 | 情緒效果 |
|----------|------|----------|
| 極遠景（ELS） | 建立場景全貌 | 孤獨、壯闊 |
| 遠景（LS） | 人物與環境的關係 | 客觀、敘述 |
| 中景（MS） | 腰部以上 | 對話、互動 |
| 近景（CU） | 面部特寫 | 情感、張力 |
| 極近景（ECU） | 局部細節 | 強調、緊張 |
| 鳥瞰（Bird's Eye） | 正上方俯拍 | 渺小、全知 |
| 低角度（Low Angle） | 由下往上拍 | 威權、力量 |
| 高角度（High Angle） | 由上往下拍 | 弱小、脆弱 |

---

## 分鏡（Storyboard）要素

- 鏡頭編號與場景說明
- 畫面構圖（九宮格法則）
- 攝影機運動方式（推、拉、搖、移、跟）
- 角色位置與動作
- 對話／旁白
- 預估時間長度

---

## 導演溝通重點

- **與攝影師（DOP）**：確認視覺風格、燈光方向、鏡頭選擇
- **與演員**：建立信任、討論角色動機而非指令動作
- **與剪接師**：預先溝通節奏感與剪接邏輯
- **與製片**：控管時程、預算與現場執行效率

---

## 視覺風格參考方向

- **色調（Color Grade）**：暖調 / 冷調 / 去飽和 / 高對比
- **畫面比例**：2.39:1（電影寬銀幕）/ 1.85:1 / 16:9（電視/串流）
- **攝影機運動**：手持搖晃（紀實感）/ 穩定器（流暢）/ 固定機（靜謐）
