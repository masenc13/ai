### 主要平台內容投資

| 平台 | 2025 年內容預算 | 重點策略 |
|------|----------------|----------|
| **Netflix** | **USD 180 億** | 吸引好萊塢導演、原創系列、跳過院線 |
| **Amazon Studios** | 未公開（數十億美元級） | 原創電影 + Prime 串流 |
| **Apple TV+** | 未公開 | 小而精、品質導向 |
| **Disney+** | 與 Hulu 整合、精簡開支 | IP 驅動（Marvel、Star Wars） |

### Netflix 對導演的吸引力

- 創意自由度高：支持不易商業化的題材與高預算實驗
- 跳過院線：省去行銷費用，如《電子國度》製作費達 **USD 3.2 億**，仍選擇串流首播
- 訂閱用戶超過 **3 億**，觸及規模遠超傳統院線
- 2025 年起積極引進知名導演（Russo Brothers、Alfonso Cuarón 等）

### 需求最高的導演技能（2025）

1. **串聯敘事（Serialized Storytelling）**：Netflix、Disney+ 等平台大量需要能駕馭多集格局的導演
2. **垂直影片製作**：TikTok / Reels 格式的短影音敘事
3. **AI 輔助後期製作理解**：AI 自動剪輯、色調調整、視覺特效
4. **跨文化內容製作**：韓劇、西班牙語系、印度內容全球化需求

## 四、影視製作關鍵趨勢

### AI 在影視製作的應用

- **AI 自動剪輯**：粗剪、節奏建議
- **AI 視覺特效（VFX）**：降低後期成本
- **個人化推薦演算法**：決定內容曝光，影響內容創作方向
- **AI 生成內容（AIGC）**：場景概念圖、分鏡預視（Previs）

### 區域市場成長

| 區域 | 特點 |
|------|------|
| 北美 | 最大市場，Netflix / Amazon / Apple 主導 |
| 亞太 | 韓國、印度、日本原創內容崛起，串流高速成長 |
| 非洲 | 預期成長最快的新興市場 |

### 製作規格趨勢

| 規格 | 現況 |
|------|------|
| **4K HDR** | 串流主流交付規格 |
| **杜比全景聲（Dolby Atmos）** | 高端串流內容音效標配 |
| **垂直影片（9:16）** | 社群短影音必備格式 |
| **虛擬製片（Virtual Production）** | LED 牆取代傳統外景，降低成本與碳排 |
