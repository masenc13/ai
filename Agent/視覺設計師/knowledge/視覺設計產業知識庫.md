## 三、產業趨勢與挑戰

### 關鍵趨勢

1. **AI 輔助設計流程**：AI 處理重複性工作（素材生成、初稿），設計師聚焦決策與策略思維
2. **雲端協作主流化**：雲端部署佔 UX 設計市場 **65.1%**（CAGR 17.45%）
3. **空間運算 / AR/VR**：隨 Apple Vision Pro 等裝置普及，空間介面設計需求快速成長
4. **設計系統（Design System）**：企業大規模採用，確保跨產品視覺一致性
5. **無障礙設計（Accessibility）**：WCAG 規範要求提升，成為設計標配

### 人才市場

- **76%** 的企業反映難以招募 UI/UX 人才（Experis UK, 2025）
- 德國 UI/UX 人才缺口從 **28% 急升至 61%**（單年內）
- 設計師職能擴大：需具備 AI prompt 能力、資料解讀、商業策略理解

## 四、設計交付標準與評量指標

### 使用者體驗 KPI

| 指標 | 說明 |
|------|------|
| **任務完成率（Task Completion Rate）** | 使用者成功完成特定任務的比例 |
| **錯誤率（Error Rate）** | 操作過程中發生錯誤的頻率 |
| **學習效率（Time on Task）** | 完成任務所需時間 |
| **系統可用性量表（SUS Score）** | 0–100 分，>68 為可接受，>80 為優秀 |
| **淨推薦值（NPS）** | 使用者推薦產品的意願（-100 到 +100） |

### 設計投資回報

- 每投入 USD 1 於 UX，平均回報 USD 100（ROI 9,900%）（Forrester Research）
- 良好的 UX 設計可將轉換率提升最高 **400%**（Forrester）
