# 動態設計與影音廣告視覺知識庫（視覺設計師視角）

> 聚焦動態設計（Motion Graphics）、廣告影片視覺製作、品牌動態識別系統。
> 資料截止日期：2025–2026 年。

---

## 一、動態設計基礎概念

### 什麼是 Motion Graphics

Motion Graphics（動態設計）是將靜態視覺元素（文字、圖形、Logo）賦予時間與動態的設計學科。在廣告影片中，動態設計扮演：
- **品牌識別的動態延伸**：Logo 動畫、品牌色彩的動態呈現
- **資訊視覺化**：產品規格、數據圖表動畫
- **敘事輔助**：字幕、Callout、Transition 效果
- **廣告結尾識別（End Card）**：統一品牌收尾，導引 CTA

## 二、動態設計的核心原則

### 動態設計的 12 個動畫原則（源自迪士尼）

| 原則 | 說明 | 廣告設計應用 |
|------|------|-------------|
| **Squash and Stretch（壓縮與拉伸）** | 物體運動時的彈性變形 | Logo 彈跳動畫 |
| **Anticipation（預期動作）** | 動作前的預備動作 | 按鈕點擊前的輕微縮放 |
| **Staging（舞台感）** | 引導視覺注意力到重點 | 重要資訊的入場動畫 |
| **Straight Ahead / Pose to Pose** | 逐格繪製 vs 關鍵格插值 | AE 關鍵格設定方式 |
| **Follow Through（跟隨與重疊）** | 主體停止後附屬物持續運動 | 文字入場後的細節殘留 |
| **Slow In / Slow Out（緩入緩出）** | 動作開始慢、結束慢 | 所有 UI 動畫的基礎 Easing |
| **Arcs（弧線運動）** | 自然的弧形路徑 | 自然流暢的元素移動 |
| **Secondary Action（次要動作）** | 強化主動作的輔助動作 | 圖形展開同時文字淡入 |
| **Timing（時間感）** | 動作速度決定情緒 | 快速剪輯 vs 慢動作情緒 |
| **Exaggeration（誇張）** | 超過現實的動態強化效果 | 產品英雄時刻的動態特效 |
| **Solid Drawing（立體感）** | 3D 空間感 | 2.5D / 3D 元素設計 |
| **Appeal（吸引力）** | 整體視覺魅力 | 品牌角色、吉祥物動畫 |

### Easing（緩動曲線）類型

| 類型 | 說明 | 使用時機 |
|------|------|----------|
| **Linear** | 等速運動 | 機械感、數據視覺化 |
| **Ease In** | 由慢到快 | 元素離場 |
| **Ease Out** | 由快到慢 | 元素入場（最常用）|
| **Ease In-Out** | 慢→快→慢 | 元素從A到B的移動 |
| **Overshoot（Back）** | 超出目標後回彈 | 強調元素、Logo 動畫 |
| **Spring** | 彈簧效果 | 互動 UI、可愛品牌風格 |

## 三、廣告影片視覺設計的核心元素

### 1. Logo 動畫（Logo Animation）

Logo 動畫是品牌在影片中最重要的識別資產，通常出現在：
- **影片開頭（Intro）**：建立品牌背書，通常 1.5–3 秒
- **影片結尾（Outro / End Card）**：最後印象，通常 2–5 秒
- **Bumper Ad**：6 秒廣告中 Logo 必須在前 2 秒出現

**Logo 動畫設計原則：**
- 保持 Logo 比例一致，不過度變形
- 動態速度與品牌調性匹配（快速活潑 vs 緩慢高端）
- 在深色和淺色背景都需有版本
- 輸出格式：透明背景 MOV（ProRes 4444）/ 含 Alpha Channel

### 2. Lower Third（下方字幕條）

用於影片中人名、職稱、地點等資訊的動態字幕設計。

**設計規範：**
- 安全區域：置於畫面下方 20% 以內
- 背景：品牌色半透明色塊，確保可讀性
- 動畫：從左側滑入（最常見）/ 漸變顯示
- 字體大小：至少 30pt（1080p 畫面），確保不同螢幕可讀
- 停留時間：至少 3–4 秒（讓觀眾有時間閱讀）

### 3. End Card（影片結尾卡）

廣告影片最後 3–10 秒的固定版面，通常包含：
- 品牌 Logo（動態版本）
- Slogan / 核心訊息
- CTA（Call to Action）：網址、App Store 連結、電話
- QR Code（視情況）
- 社群帳號

**End Card 設計注意事項：**
- YouTube End Screen：最後 20 秒，可疊加互動元素
- 靜態 End Card vs 動態 End Card：靜態更易製作，動態更有品牌感
- 確保 CTA 文字大小在手機螢幕可讀

### 4. Supers（字幕疊加）

廣告中疊加在畫面上的文字訊息：
- **法律聲明（Legal Supers）**：「詳見官網條款」等小字，通常在畫面底部
- **產品資訊 Supers**：價格、規格、限時優惠
- **對話字幕（Captions）**：無聲播放時的關鍵字幕

## 四、廣告影片色彩管理

### 廣播安全色（Broadcast Safe Colors）

電視廣告的色彩必須符合廣播標準，超出範圍會造成色彩失真：

| 色彩空間 | 規格 | 用途 |
|----------|------|------|
| **Rec. 709** | HD 電視廣播標準色域 | 台灣電視廣告、一般串流 |
| **Rec. 2020** | UHD / HDR 擴展色域 | 4K HDR 內容 |
| **sRGB** | 數位螢幕標準 | 社群平台、網頁 |

**廣播安全色範圍（Rec. 709）：**
- 亮度（Luma）：16–235（8-bit）
- 彩度（Chroma）：不可超飽和

### 品牌色彩在影片中的一致性

- 從 **RGB（螢幕顯示）** 轉換為 **Hex / Pantone** 時需注意色彩模式差異
- 印刷品的 CMYK 色值在螢幕上顯示可能偏暗
- 使用 DaVinci Resolve 的色彩管理功能統一跨鏡頭的品牌色還原
- 建立 **LUT（Look Up Table）**：品牌專用的色彩調校預設，確保跨影片一致性

## 六、廣告影片尺寸規格全覽

### 橫式影片（16:9）

| 平台 / 用途 | 解析度 | 幀率 | 格式 |
|------------|--------|------|------|
| 電視廣告（HD） | 1920×1080 | 25fps | ProRes 422 HQ |
| YouTube 廣告 | 1920×1080 / 3840×2160 | 23.976 / 25 / 30fps | H.264 / HEVC |
| Facebook Video Ad | 1920×1080 | 24–30fps | MP4（H.264）|
| LinkedIn Video | 1920×1080 | 最高 30fps | MP4 |

### 直式影片（9:16）

| 平台 / 用途 | 解析度 | 最大時長 | 格式 |
|------------|--------|----------|------|
| TikTok 廣告 | 1080×1920 | 60s（最佳 15–30s） | MP4 |
| Instagram Reels 廣告 | 1080×1920 | 90s | MP4 |
| YouTube Shorts 廣告 | 1080×1920 | 60s | MP4 |
| Instagram Stories 廣告 | 1080×1920 | 15s | MP4 |

### 方形影片（1:1）

| 平台 / 用途 | 解析度 | 說明 |
|------------|--------|------|
| Instagram Feed 廣告 | 1080×1080 | 停留在 Feed 最大面積 |
| Facebook Feed 廣告 | 1080×1080 | 跨桌機 / 手機通用 |

### 安全區域（Safe Zone）設計原則

在所有影片中，重要元素需置於「安全區域」內：
- **標題安全區（Title Safe）**：距畫面邊緣 5%
- **行動安全區（Action Safe）**：距畫面邊緣 3.5%
- **9:16 特別注意**：上下各有 UI 遮擋（手機狀態列、互動按鈕），關鍵元素置於中間 70% 區域

## 七、廣告影片字體設計規範

### 影片中字體使用原則

| 規則 | 說明 |
|------|------|
| **最小字體大小** | 1080p 畫面：至少 28pt；手機播放：至少 36pt |
| **行距** | 行高至少 1.4 倍字高，確保螢幕可讀性 |
| **對比度** | 文字與背景對比度至少 4.5:1（WCAG AA 標準）|
| **陰影/描邊** | 白色文字加深色陰影，確保在亮色背景可讀 |
| **字重** | 影片字幕建議 Medium（500）至 Bold（700）|

### 廣告字幕（Captions）設計

- **社群平台 80% 無聲觀看**：字幕設計成為廣告成效關鍵
- 字幕位置：畫面下方 20–30% 區域，避開重要視覺
- 字幕風格：**動態字幕（Kinetic Typography）** 比靜態更吸引注意
- 字幕格式：SRT 檔案（外掛字幕）vs 燒入字幕（Burned-in Captions）

## 八、品牌動態識別系統（Motion Branding）

品牌動態識別系統（Motion Identity）是將靜態品牌視覺系統延伸至動態媒體的系統規範。

### 核心元素

| 元素 | 說明 |
|------|------|
| **Logo 動畫** | 品牌標誌的動態版本，含時間與動態規範 |
| **品牌動態色彩** | 色彩的漸變方式、轉場應用 |
| **轉場動畫（Transitions）** | 場景切換的品牌化方式（如特定顏色遮罩）|
| **品牌字體動態規範** | 文字入場方式、動態速度 |
| **音效識別（Audio Branding）** | Logo Sting（品牌聲音識別）、背景音樂風格指引 |
| **動態設計語言文件** | 供所有設計師 / 後期人員參考的 Motion Guidelines |

### 建立 Motion Guideline 的步驟

1. 確立品牌個性關鍵字（如：精準、溫暖、活力）
2. 定義對應的動態速度（快/慢）和 Easing 類型
3. 制定 Logo 動畫使用規範（時長、不得超過的變形程度）
4. 建立核心動態範本（After Effects 模板）
5. 製作示範影片，提供給製作公司參考

## 十、企業形象影片的字卡與視覺風格設計規範

> 本節知識源自藍麒科技_貿協提案用腳本的實務分析。適用於 B2B 企業品牌影片的視覺設計提案。

### 字卡（Supers）設計策略

企業形象影片的字卡不是每段都需要，應有選擇地使用：

| 字卡類型 | 使用時機 | 設計重點 |
|----------|----------|----------|
| **歷史/數字型** | 開場（強調積累與信任）| 大字號數字，簡潔有力，如「20年」|
| **品牌聲明型** | 競爭優勢段落 | 精簡一句話，如「台灣唯一使用德國巴斯夫原料」|
| **技術規格型** | 工藝/技術段落 | 規格數字突出，如「12米一體成型」|
| **認證標準型** | 願景/市場段落 | 配合應用場景同步出現，如「ISO認證」|
| **Slogan + LOGO** | 品牌精神結尾 | 全畫面靜止，Slogan居中，LOGO動畫收尾 |

**字卡設計原則：**
- 純功能段落（品管流程、服務能力）**不放字卡**，讓畫面與VO說話
- 字卡應該是對VO的「視覺強調」，而非重複說明
- B2B影片字卡語言須精準、去掉形容詞，多用數字與專有名詞

---

### 企業形象影片視覺風格定義

提案前需確立的 4 個視覺維度：

| 維度 | 選項 | B2B科技製造業建議 |
|------|------|-----------------|
| **色調（Color Tone）** | 冷色調 / 暖色調 / 中性 | 冷色調（藍灰鐵器感）＋局部暖光（人物情感段落）|
| **畫面比例** | 16:9 / 2.39:1（電影感）| 16:9為主，關鍵技術段落可用 2.39:1 letterbox 強調電影感 |
| **節奏感** | 快剪 / 慢動作 / 混合 | 技術段落慢動作特寫，中段快剪帶動節奏感 |
| **燈光方向** | 自然光 / 工業光 / 棚拍 | 工廠現場工業光，研發/品管場景需補光提升清晰度 |

---

### 各段落的視覺設計重點

| 段落 | 視覺設計師關注重點 |
|------|--------------------|
| **開場** | 產品主體置中，慢動作帶出細節感，數字字卡字體選用 Sans-Serif 粗體 |
| **材料/技術段** | 素材剪輯的色調統一，避免素材色溫不一致 |
| **實測段** | 熱顯像/數據畫面的色彩處理（偽色彩顯示），建立視覺驚喜 |
| **工廠段** | 空拍素材的穩定器處理，機台特寫需補工業藍光 |
| **人物/服務段** | 人物色溫統一至日光白，情緒相對溫暖 |
| **市場/願景段** | 三個不同應用場景的色調應有輕微差異（倉儲/醫療/半導體各有情境感） |
| **品牌收尾** | LOGO動畫需提前規劃透明版（ProRes 4444），與畫面合成的timing需精確 |
