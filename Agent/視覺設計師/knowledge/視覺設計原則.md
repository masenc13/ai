# 視覺設計師知識庫

## 設計基本原則

- **對齊（Alignment）**：所有元素應有視覺上的連結，避免隨意擺放。
- **對比（Contrast）**：透過顏色、大小、字重的差異，引導視覺焦點。
- **重複（Repetition）**：統一視覺風格，強化品牌一致性。
- **近接（Proximity）**：相關元素應靠近擺放，讓版面清晰易讀。

## 色彩理論

| 類型 | 說明 |
|------|------|
| 主色（Primary） | 品牌核心識別色，用於最重要的視覺元素 |
| 輔助色（Secondary） | 搭配主色使用，增加層次 |
| 強調色（Accent） | 用於 CTA、高亮區塊，佔比不超過 10% |
| 中性色（Neutral） | 背景、文字、分隔線 |

**色彩情緒參考：**
- 紅：熱情、緊迫、力量
- 藍：信任、穩定、專業
- 黃：樂觀、創意、活力
- 綠：自然、成長、健康
- 紫：奢華、創意、神秘

---

## 字體排版

- 標題字體：選擇具識別性的字體，字重 700+
- 內文字體：優先選用高可讀性字體（如 Noto Sans、Inter）
- 行距建議：內文行距為字級的 1.5～1.75 倍
- 中文字體推薦：思源黑體、思源宋體、微軟正黑體
